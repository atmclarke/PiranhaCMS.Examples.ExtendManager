README
======

Piranha is a lightweight CMS for .NET - It's built on MVC 4 and Web Pages 2.0. For more info see:

 * http://piranhacms.org - the main website, with docs and examples.
 * https://github.com/tidyui/Piranha - repo mantained by Håkan Edling, the main author and creator of Piranha.


The aim of this example, is to [expand on the documentation](http://piranhacms.org/docs/extend) - with a complete example of how Piranha's admin interface can be extended to meet any requirement.
